# Azure Security Labs with Terraform  

This repository contains infrastructure-as-code (IaC) labs that demonstrate modern, secure cloud architectures in Microsoft Azure, implemented with Terraform.  

|                       Lab                       |                                                              Description                                                             | Folder |
|:-----------------------------------------------:|:------------------------------------------------------------------------------------------------------------------------------------:|:------:|
| Hub-and-Spoke Network with Azure Firewall       | Enterprise-grade network topology with centralized egress inspection,  VNet peering, Bastion access, and UDR-based traffic steering. |        |
| Sentinel Threat Detection: Brute-Force Sign-ins | Production-safe Microsoft Sentinel analytics rule that detects suspicious  login attempts and creates incidents                      |        |
|                                                 |                                                                                                                                      |        |
